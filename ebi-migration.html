--- layout: idr title: IDR EBI migration ---

The Image Data Resource is currently running with reduced functionality

During March 2020 the EMBL-EBI data centre which hosts the IDR is migrating to a new physical location. From 11th March 2020 until roughly the end of the month the following IDR services will be unavailable:

  • Virtual Analysis Environment (VAE)
  • FTP submissions

Web and API access to the IDR will continue to function, thanks to the provision of a temporary data centre.

See @IDRstatus for updates
