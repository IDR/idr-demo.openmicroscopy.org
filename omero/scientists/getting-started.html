Getting Started

Getting started with OMERO is easy with our range of ‘How to’ guides. If you have an institutional server to connect to, you can get up and running in minutes. If OMERO is all new to you, we have a demo server you can try out.
Try out the Demo server

Register and we’ll send you a log-in to try out OMERO before committing any IT resources to setting up your own server. This guide explains the terms of use and takes you through connecting to the server via the desktop and web-based apps.
