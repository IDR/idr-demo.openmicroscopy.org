OMERO is designed with scientists in mind

From the microscope to publication, OMERO handles all your images in a secure central repository. Upload your data via the desktop app or command line, use our custom dropbox tool, or have your facility manager import it for you. Once securely stored, you can view, organize, analyze and share your data from anywhere you have internet access. Work with your images from a desktop app (Windows, Mac or Linux), from the web or from 3rd party software; scroll through Z-sections and time points, turn channels on and off, adjust rendering settings, add comments, ratings and tags. OMERO integrates with popular image analysis tools such as ImageJ and MATLAB and features a scripting service, allowing you to run analysis on the server and save the results as attachments to your images. No more emailing images to your collaborators or PI, instead you can share data via URLs or even publish them direct from the server via a customized website or embedded viewer. Using the power of Bio-Formats, OMERO supports over 140 image file formats, including all major microscope formats.

Key Features

OMERO has many features designed with scientists in mind. Some of the key ones are highlighted here.
Import Data

Import over 140 image formats from microscope, graphics and other imaging formats, using the OMERO.insight client.

View User Guide
View Data

Use OMERO.insight and OMERO.web on any platform to view and work with image data over the internet from anywhere. Browse thumbnails, image previews and view in full viewer including 'split-channel' view and Z-projection.

View User Guide
Rendering Settings

Rendering settings, including colours and intensity ranges can be adjusted and saved for any image. In appropriate groups, other group members can save their own settings for shared images.

View User Guide
Create ROIs

The Measurement Tool in OMERO.insight is used to draw Regions of Interest (ROIs) and enable some basic analysis to be performed.

View User Guide
Add Metadata

Add tags, key-value pairs, comments, ratings or file attachments to single or multiple images or containers. Search sort or interact with data based on these annotations.

View User Guide
Draw Figures

Use OMERO.figure to draw figures quickly and easily for publications, presentations and meetings, save them on the OMERO server, and export as vector graphics.

View User Guide
Collaborate

OMERO Groups enable sharing of data, annotations and figures within groups of scientists. Browse multiple groups and move data between them. Group owners have extra capabilities

View User Guide
Image/Fiji Integration

The OMERO ImageJ plugin enables images on OMERO to be opened directly in ImageJ/Fiji. Save ROIs and overlays, results, new images or modified images from ImageJ into OMERO.

View User Guide
Export As

Data stored on the OMERO server can be exported in the original format or as an OME-TIFF, PNG, JPEG or TIFF, singly or as batches.
